# Contributing to the Company Brain Benchmark

The methodology is opinionated, but the scoring is contestable. Open an issue.

## Ways to contribute

### Contest a score

Open an issue using the **Contest a score** template. Include:

1. Product name
2. Dimension (Freshness / Audience-safety / Portability / Provenance)
3. Current score
4. Score you think it should be
5. Public evidence (URL to docs, engineering blog, your hands-on test, etc.)

We review weekly. Substantive evidence updates the score. The changelog records who pushed back and why.

### Add a product

Open an issue using the **Add a product** template. Include:

1. Product name
2. 1-paragraph architectural summary
3. Why it positions as a "company brain" / knowledge platform / AI knowledge layer

We add new products quarterly. We do **not** score every memory library, vector DB, or workflow framework — only products that publicly position themselves at the company-brain layer.

### Propose a methodology change

Open an issue using the **Methodology change** template. Cover:

1. The problem with the current rubric
2. The proposed change (new dimension, modified rubric, removed criterion)
3. Why this makes the benchmark more useful for buyers / builders

Methodology changes increment the version (e.g., `v2026.04` → `v2026.07`). Per-product rescores happen automatically.

## Scoring principles

- **Public evidence only.** Engineering blogs, official docs, public pricing, hands-on testing. No secondary reporting.
- **Honest scoring of self.** We score BizBrain OS first because we wrote the rubric. Where we lose, we say so.
- **No "coming in Q3" scoring.** We score products as they are, not as they're promised.
- **No popularity bias.** Distribution and customer count matter for buyers, but they're not in this rubric. Different question.

## What this benchmark is NOT

- A buyer's guide. We don't tell you which product to buy. We tell you what to evaluate against.
- A funding signal. We don't list valuations or ARR.
- A leaderboard for community voting. The methodology is fixed; the scores are evidence-based.
